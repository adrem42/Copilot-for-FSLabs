Installation

  1. Drop the Modules folder into the sim root folder.

  2. Edit FSUIPC5.ini to add:

    [Auto.FSLabs]
    x=Lua FSLabs Copilot

    where x is 1 or the next number in sequence if the [Auto.FSLabs] section already exists and contains - entries.

    Replace ‘FSLabs’ in ‘Auto.FSLabs’ with the name of your FSLabs-specific FSUIPC profile if you have one.

  3. Load an FSLabs aircraft once to let the script create the options file.

  4. Edit the options file at sim root/Modules/FSLabs Copilot/options.ini

Setting up speech recognition

You need to have English as the Windows language in order for speech recognition to work

  1. Go to Control Panel -> Speech Recognition -> Advanced speech options

  2. Select English - UK or English - US as the language

  3. Select Configure Microphone in the Microphone section

  4. It’s essential to train the profile with the Train Profile wizard
