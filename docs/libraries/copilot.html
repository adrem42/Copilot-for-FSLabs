addCallback(func, name)
adds function or couroutine func to the main callback loop

Parameters:

  • func a function or thread
  • name optional string which can be used to remove the callback with removeCallback

Returns:

    func
removeCallback(key)
removes a previously added callback

Parameters:

  • key Either the function or thread itself or the name argument passed to addCallback
