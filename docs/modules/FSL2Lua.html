Tables

FSL
The main module table containing all the controls and some other functions

Fields:

  • CPT table containing controls on the left side
  • FO table containing controls on the right side
  • PF table containing controls on the side of the Pilot Flying
    For example, if the PF is the Captain, it will be the same as the CPT table.
    The controls on the PM side are in the root FSL table.

Usage:

    FSL.GSLD_EFIS_VORADF_1_Switch("VOR")

Class MCDU

MCDU:getArray()

Returns:

    table: array of tables representing display cells.

    Each cell table has three fields:

    char: the character (nil if the cell is blank)

    color: string :

    • ‘green’
    • ‘magenta’
    • ‘amber’
    • ‘white’

Class Control

Abstract control
Control:macro(clickType)
If for some reason you want to use a macro directly instead of using one of the control’s methods

Parameters:

  • clickType string: One of the following:
    • ‘leftPress’
    • ‘leftRelease’
    • ‘rightPress’
    • ‘rightRelease’
    • ‘wheelUp’
    • ‘wheelDown’
Control:isLit()

Returns:

    boolean : true if the control has a light and it’s on.

    The control needs to have an LVar associated with its light for this to work.

    Unfortunately, overhead-style square buttons don’t have such LVars.

Usage:

    if not FSL.GSLD_EFIS_CSTR_Button:isLit() then
  FSL.GSLD_EFIS_CSTR_Button()
end
Control:getLvarValue()

Returns:

    number

Class KnobWithoutPositions

Knobs with no fixed positions
KnobWithoutPositions:__call(targetPos)

Parameters:

  • targetPos number: relative position from 0-100

Usage:

    FSL.OVHD_INTLT_Integ_Lt_Knob(42)
KnobWithoutPositions:rotateLeft()
rotate the knob left by 1 tick
KnobWithoutPositions:rotateRight()
rotate the knob right by 1 tick
KnobWithoutPositions:getPosn()

Returns:

    number: relative position from 0-100
KnobWithoutPositions:rotateBy(ticks[, pause=70])
Rotates the knob by amount of ticks

Parameters:

  • ticks number positive to rotate right, negative to rotate left
  • pause number milliseconds to sleep between each tick (default 70)
KnobWithoutPositions:random([lower=0[, upper=100]])
Set to random position between lower and upper

Parameters:

  • lower number (default 0)
  • upper number (default 100)

Bind(data)

Parameters:

  • data table

    For a joystick button bind, define the ‘btn’ field, the value of which should be a string containing the FSUIPC joystick letter and button number (for example, ‘A42’).

    For a key bind, define the ‘key’ field.

    The following values for ‘key’ are accepted:

    • Alphanumeric character keys
    • Escaped keycode for weird characters like ‘ö’ (key = ‘\222’).
      The keycodes can be looked up in the FSUIPC console after enabling the ‘Button and key operations’ logging facility.
    • Enter
    • Pause
    • CapsLock
    • Esc
    • Escape
    • Space
    • PageUp
    • PageDown
    • End
    • Home
    • LeftArrow
    • UpArrow
    • RightArrow
    • DownArrow
    • PrintScreen
    • Ins
    • Insert
    • Del
    • Delete
    • NumpadEnter
    • NumpadPlus
    • NumpadMinus
    • NumpadDot
    • NumpadDel
    • NumpadDiv
    • NumpadMult

    You can combine one key with one or more of the following shift keys using + as the delimiter:

    • Tab
    • Shift
    • Ctrl
    • LeftAlt
    • RightAlt
    • Windows
    • Apps

Usage:

    Bind {key = "SHIFT+A", onPress = FSL.MIP_ISIS_BARO_Button}
Bind {
  key = "NumpadEnter",
  onPressRepeat = function() FSL.OVHD_INTLT_Integ_Lt_Knob:rotateLeft() end
}
Bind {
  btn = "C5",
  onPress = FSL.PED_COMM_INT_RAD_Switch_RAD,
  onRelease = FSL.PED_COMM_INT_RAD_Switch_OFF
}
